Cisco Talos documents ARToken, a sophisticated phishing-as-a-service (PhaaS) operator panel affiliated with the EvilTokens platform, which exposes 80+ API endpoints enabling device code phishing, Primary Refresh Token persistence, email compromise, BEC operations, and SharePoint exfiltration—indicating a mature operational environment rather than a basic phishing kit. The intelligence includes IOCs for defensive hunting and blocking. The newsletter also covers concurrent threats: an 81-million-attempt password-spraying campaign against Microsoft 365 using breached credentials, critical exploitation of SimpleHelp (CVE-2026-48558) authentication bypass for malware delivery, and growing abuse of misconfigured AI endpoints by threat actors to power sophisticated, evasive attacks.
