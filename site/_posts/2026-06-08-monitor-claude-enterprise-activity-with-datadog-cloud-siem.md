Datadog released a Claude Compliance API integration that normalizes Claude Enterprise activity events (sign-ins, admin actions, API key lifecycle, config changes) into standardized OCSF schema for centralized security monitoring. The integration includes prebuilt Cloud SIEM detection rules to flag suspicious events (unexpected admin changes, unusual API key creation, suspicious auth patterns) and correlate them with other security telemetry. Compliance teams gain dashboards for activity overview, geographic login anomaly detection, and exploratory analysis across actors, source IPs, and event types—enabling consistent AI-related activity monitoring across organizations.
