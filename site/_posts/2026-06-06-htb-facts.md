A HackTheBox write-up demonstrating exploitation of a Ruby on Rails-based trivia site (Camaleon CMS) via mass assignment vulnerability to escalate to administrator, then pivoting through leaked MinIO credentials and a cracked SSH key. Privilege escalation abuses a misconfigured sudo rule on Puppet's facter tool to load arbitrary Ruby code as root. Beyond Root section covers path traversal in Camaleon's S3 uploader to read arbitrary files and Rails master key decryption to access encrypted credentials and session cookies.
