Wiz's M-Red-Team documented a supply chain compromise of the AsyncAPI project exploiting GitHub Actions to inject malicious code into the build pipeline. The attack demonstrates how misconfigured CI/CD workflows can be weaponized to compromise widely-distributed open-source packages, affecting all downstream consumers. This represents a critical CI/CD attack vector targeting the automation and package distribution layer. The research illustrates the gap between GitHub Actions security defaults and the trust model of open-source ecosystems.
