CTF write-up exploiting ZoneMinder v1.37.63 via CVE-2024-51482, a boolean-based SQL injection in the event tag removal endpoint (`web/ajax/event.php`). The attack chains multiple vulnerabilities: extracting user credentials via blind SQLi in the `tid` parameter, cracking bcrypt hashes for SSH access, sniffing cleartext passwords with tcpdump, and leveraging authenticated command injection in motionEye's still-image filename setting to achieve root code execution. The write-up also documents an unintended path via Motion's unauthenticated control interface for direct privilege escalation.
