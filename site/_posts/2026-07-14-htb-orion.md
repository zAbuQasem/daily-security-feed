The Orion HTB writeup documents exploitation of an unauthenticated remote code execution vulnerability in Craft CMS's image transform endpoint. The attack chains object injection in the underlying Yii Framework to poison a PHP session file, achieving code execution as the web user. After gaining access, database credentials are extracted from Craft's config files, enabling credential harvesting and privilege escalation through bcrypt hash cracking. The final stage exploits an authentication bypass in an inetd-hosted telnet service via USER environment variable smuggling to achieve root access. The writeup demonstrates practical exploitation of object injection vulnerabilities in PHP/Yii applications and environment-based authentication bypass techniques relevant to real-world PHP deployments.
