Microsoft's June 2026 Patch Tuesday addresses 206 vulnerabilities including 32 critical RCEs. Notable flaws include heap-buffer overflows in Remote Desktop Client and DHCP Client, an integer overflow in HTTP.sys (CVE-2026-47291) exploitable by unauthenticated attackers, a use-after-free in Windows Kernel (CVE-2026-45657) enabling remote code execution with system privileges, type-confusion bugs in Office/Outlook/Word exploitable via email preview, and Hyper-V guest-to-host escapes. Affected components span Windows Graphics, Active Directory, Kerberos KDC, WDS, SQL Server, and Azure—all requiring immediate patching.
