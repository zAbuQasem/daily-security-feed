SonarQube Server 2026.4 introduces an "Agentic AI" quality gate calibrated for AI-generated code risk: stricter on security, reliability, and supply chain issues (typosquatted/hallucinated packages), more permissive on style. New security rules target CLI injection surfaces, MCP-based threats, and data leaks unique to agent codebases. Architecture management now available on Server (previously Cloud-only) to flag structural drift; taint graph optimizations deliver up to 90% faster analysis for large codebases.
