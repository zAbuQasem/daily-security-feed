Analysis of Apple's October 2025 Security Bounty program changes, which reduced TCC vulnerability payouts by 83.6% (from $30.5k to $5k), narrowed Gatekeeper scope to require Safari downloads and quarantine bits (invalidating prior bypasses), and cut App Sandbox escape rewards by 52.4% ($10.5k to $5k). The author examines community backlash and questions whether these changes represent cost-cutting, providing context for researchers and discussing the program's fairness and long-term trajectory. Valuable for understanding how incentive structures shape the bug bounty and vulnerability disclosure landscape.
